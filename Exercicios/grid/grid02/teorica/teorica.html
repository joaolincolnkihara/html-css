GRID

A GRID e dividida em linhas e colunas como uma grelha.

Quando o container e "display: grid;" os itens ficam "display: block;"

display: grid;

//CONTAINER

GRID-TEMPLATE - define as colunas e as linhas.

Vc ja coloca a largura com a quantidade de colunas ou linhas.

fr - significa fraçã, logo 1fr é uma fração.

grid-template-columns: Xpx Xfr X%;

grid-template-rows: X% Xfr Xpx;

grid-template-areas: ;

GAP - É o espaço entre colunas/linhas.

gap: row-gap;

gap: column-gap;

//ITENS

O "Start" começa na primeira linha virtual e o "End" termina na ultima linha virtual.

A short hand dos grid-start/end são "grid-column: x/y;" e "grid-row: x/y;"

grid-coumn-start: ;

grid-coumn-end: ;

grid-row-start: ;

grid-row-end: ;

O "grid-area" vai trabalhar em conjunto geralmnte com o "grid-template-areas"

É usado para nomear oq vai estar no grid-template-areas

grid-area: ;

//ALINHAMENTOS

os alinhamentos como "align", "justify" e "place" funcionam igual o flex-box
