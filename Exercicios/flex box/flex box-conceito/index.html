Flexbox

FLEX BOX - CONTAINER

flex-direction

usada no "container"

flex-direction: row; = vai da direita para a esquerda
flex-direction: row-reverse; = sentido contrario do row
flex-direction: column; = no sentido de cima para baixo
flex-direction: column-reverse; = no sentido de cima para baixo

Eixos(axis)

Flex-wrap

flex-wrap: nowrap; = achata os itens quando acaba o espaço
flex-wrap: wrap; = quebra a linha quando acaba o espaço

flex-flow

Junta o flex-direction + flex-wrap
flex-flow: row nowrap;

Justify-content

Faz o alinhamento de acordo com o main-axis (eixo principal)

Aling-itens

Faz o alinhamento de acordo com o cross-axis (eixo transversal)

por padrao vem como "align-itens: stretch" (alonga o item)

align-content

Faz o alinhamento de acordo com o cross-axis (eixo transversal) quando estão enpacotados (wrap)
